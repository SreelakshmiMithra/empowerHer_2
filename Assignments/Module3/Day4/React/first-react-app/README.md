# First React App 

This project demonstrates a simple React setup using **Vite** and creating a custom React component.

## Project Description
- Created a React component named `FirstComponent` that displays a message inside an `<h2>` tag.
- Updated `App.jsx` to render `FirstComponent`.
- Used **CSS Flexbox** to center the output both vertically and horizontally.
- Followed best practices by excluding `node_modules` via `.gitignore`.

## Changes Made
- `FirstComponent.jsx`:
  - Contains an `<h2>` message: "This is my first React component".
- `App.jsx`:
  - Replaced default Vite + React template code.
  - Imported and rendered `FirstComponent`.
- `index.css`:
  - Updated body styles to center content using `display: flex`, `justify-content: center`, and `align-items: center`.
- Added `.gitignore`:
  - Excluded `node_modules/` and other unnecessary files.
