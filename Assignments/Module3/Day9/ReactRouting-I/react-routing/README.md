# React Routing Assignment

## Description
This project demonstrates client-side routing using React Router.

## Routes
- /home – Home Page
- /aboutus – About Us Page
- /todos – Fetches and displays first 10 todos
- Any other route – 404 Page Not Found

## Technologies Used
- React
- React Router DOM
- Fetch API
- CSS

## Setup Instructions
1. Install Node.js
2. Clone repository
3. Run:
   npm install
   npm start

## Rules Followed
- No <a> tags used
- SPA navigation
- BrowserRouter, Routes, Route used
