# Todo Application Using Context API

## Objective
Build a Todo application using React Context API to manage global state and avoid props drilling.

## Features
- Add Todo
- Delete Todo
- Global state using Context API
- No props drilling
- Clean component structure

## Tech Stack
- React
- Context API
- JavaScript

## Folder Structure
src/
├── components/
│   ├── Todos.jsx
│   ├── AddTodo.jsx
│   ├── TodoList.jsx
│   └── TodoItem.jsx
├── context/
│   └── TodoContext.js
├── App.js
└── index.js

## How to Run
1. Install Node.js
2. Clone the repository
3. Run:
   npm install
   npm start

## Learning Outcome
- Understanding Context API
- Global state management
- Avoiding props drilling
- Preparing for Redux-like patterns
