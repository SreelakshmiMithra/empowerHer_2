# Solving Props Drilling Using Context API

## Objective
To understand how React Context API eliminates props drilling by providing shared data to deeply nested components.

## Problem
Passing props through multiple intermediate components causes unnecessary complexity.

## Solution
Used React Context API to store shared values and allow direct access from required components.

## Technologies Used
- React
- Context API
- useContext Hook

## Folder Structure
src/
- context/AppContext.js
- components/Component1 to Component6

## How to Run
1. npm install
2. npm start
3. Open http://localhost:3000

## Learning Outcomes
- Eliminated props drilling
- Understood Provider & Consumer
- Learned when to use Context API
